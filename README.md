# Keeg Store

## 🚀 Visão Geral
E-commerce com FronEnd completo desenvolvido com React 19 + Vite, incluindo:

- 🛒 Carrinho de compras
- ⭐ Lista de desejos
- 🔍 Filtros de produtos
- 💳 Checkout integrado

## ⚙️ Como Rodar Localmente

### Pré-requisitos
- Node.js 18+
- npm 9+

```bash
# Clone o repositório
git clone https://github.com/LuizGSN/Keeg-store.git
cd Keeg-store

# Instale as dependências
npm install

# Inicie o servidor
npm run dev
